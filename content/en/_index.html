{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Crepes provides an easy-to-use web UI to offer your users events in which they can register. It was originally designed to organise real-life courses, but has since grown to a more general purpose event registration website. You can think of it like a CMS, but for events. We're currently focussing on making it more applicable for general-purpose use cases and allowing more customization. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/nielsdos/crepes" %}} We do a [Pull Request](https://github.com/nielsdos/crepes/pulls) contributions workflow on **GitHub**. New users are always welcome! {{% /blocks/feature %}} {{< /blocks/section >}}
